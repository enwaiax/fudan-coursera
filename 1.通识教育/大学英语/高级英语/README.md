# 📝高级英语 期末考试

### **Part 1 口试**

**口试范围：**

- Audio/video clips from E-learning online assignment
- Book 5 -1B, 2B, 4A, 4B, 6A, 7A, 8A
- Book 6 - 1B, 2B, 3A, 3B, 4B



**口试方法：**

- 宣布了题目过后有3分钟看材料准备，然后2-3分钟的时间发表言论。
